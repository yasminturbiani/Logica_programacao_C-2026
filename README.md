📘 Lógica de Programação em C

Repositório criado para armazenar exercícios, exemplos e projetos desenvolvidos na disciplina de Lógica de Programação utilizando a linguagem C.

O objetivo deste repositório é praticar conceitos fundamentais de programação, desenvolver raciocínio lógico e aprender a estruturar algoritmos utilizando C.

📚 Conteúdo da Disciplina

Os principais tópicos abordados incluem:

Introdução à programação

Estrutura de um programa em C

Variáveis e tipos de dados

Operadores aritméticos e lógicos

Entrada e saída de dados (printf e scanf)

Estruturas condicionais (if, else, switch)

Estruturas de repetição (for, while, do while)

Vetores e matrizes

Funções

Algoritmos básicos
